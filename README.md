## 项目结构说明

- `./mock` 用来存放mock资源的文件夹
- `./src/assets` 用来存放资源文件的文件夹（会被打包编译）
- `./src/components` 用来存放自定义组件的文件夹
- `./src/models` 用来存放数据及操作数据方法的文件夹
- `./src/pages` 用来存放界面的文件夹
- `./src/services` 用来存放服务的文件夹，如`fetch`请求
- `./src/utils` 用来存放工具类的文件夹，如`fetch`的封装类
- `./src/global.css` 公共样式
